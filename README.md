# Charging Up!
**Made by: Marvin Agonos, Cameron Hu, Yo Shium Wong, Ryan Zacharias, John Zheng**

A BCGameJam2019 submission coded in Godot.

![Image of Yaktocat](https://i.imgur.com/8HzDKtI.png)

## Features
- Four exciting minigames!
	- "Dodge the Asteroids"
	- "Pump it Up"
	- "Fly into Space"
	- "Sumo"
- Two player gameplay.
- Local multiplayer
